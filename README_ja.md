# mcp-oss-onramp (日本語版)

**AIスカウト。最初のPRを。**

`mcp-oss-onramp`は、AIエージェントが「本当に未割り当て」かつ「初心者向け」なGitHub Issueを見つけるためのMCPサーバです。

[English README](./README.md)

## ビジョン

OSSへの貢献は摩擦ゼロであるべきです。`good first issue`のようなラベルは存在しますが、実際には既に着手されていたり、放置されたPRが紐付いていることも少なくありません。このツールは、AIエージェントに「ノイズを見抜く目」を提供します。コメントや説明文に隠れたPRリンクやアサイン状況を検出し、次のコントリビューションに最適なエントリーポイントを見つけ出します。

## 特徴

- **真の空き状況検出**: 基本的なフィルタリングを超え、コメントや説明文内のリンクされたPRも検出します。
- **AI最適化データ**: LLMが効率的に処理できるよう、Issueの本文をクリーンアップし要約します。
- **オンボーディング特化**: 成功率が高く、ハードルの低いタスクを見つけるためにチューニングされています。
- **MCPネイティブ**: Gemini CLI、Claude Code、CursorなどのAIエージェントから直接利用できるように設計されています。

## 仕組み

1. **スカウト**: AIエージェントがリポジトリ名を指定してツールを呼び出します。
2. **フィルタ**: サーバがIssueとPRを取得し、それらを照合して、オープンだが未割り当ての「ゴーストタスク」を見つけ出します。
3. **レポート**: AIがユーザーに提示するための、構造化され簡潔な推奨Issueリストを返します。
